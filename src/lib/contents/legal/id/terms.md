# Ketentuan Layanan

**Tanggal Berlaku:** 7 Mei 2026

Dengan mengakses **Squishyfile**, Anda menyetujui ketentuan berikut. Harap baca dengan saksama.

---

## 1. Layanan
Squishyfile adalah alat sisi klien untuk mengompresi dan mengonversi media. Semua pemrosesan terjadi **secara lokal di browser Anda**. Kami tidak mengunggah, menyimpan, atau memiliki akses ke file Anda. Kinerja sangat bergantung pada perangkat keras perangkat Anda.

## 2. Privasi & Data
Karena Squishyfile beroperasi sepenuhnya di mesin Anda:
* File Anda tidak pernah meninggalkan perangkat Anda.
* Kami tidak mengumpulkan data pribadi dari file yang Anda proses.
* Anda memegang kepemilikan dan tanggung jawab penuh atas konten Anda.

## 3. Penggunaan yang Dapat Diterima
Anda bertanggung jawab atas legalitas file yang Anda proses. Anda setuju untuk tidak menggunakan layanan untuk tujuan yang melanggar hukum atau melanggar hak kekayaan intelektual pihak ketiga mana pun.

## 4. Penafian Jaminan
Squishyfile disediakan **"SEBAGAIMANA ADANYA"** tanpa jaminan apa pun. Kami tidak menjamin bahwa:
* Layanan akan bebas kesalahan atau tidak terganggu.
* Kualitas output akan memenuhi persyaratan tertentu.
* Proses tidak akan mengakibatkan kerusakan data.

**Selalu simpan cadangan file asli Anda.**

## 5. Batasan Tanggung Jawab
Squishyfile dan pengembangnya tidak bertanggung jawab atas kerugian apa pun (termasuk kehilangan data atau ketegangan perangkat keras) yang timbul dari penggunaan atau ketidakmampuan untuk menggunakan layanan ini. Penggunaan alat ini merupakan risiko Anda sendiri.

## 6. Perubahan Ketentuan
Kami berhak mengubah ketentuan ini kapan saja. Terus menggunakan layanan merupakan penerimaan atas ketentuan yang diperbarui.
