# Kebijakan Privasi

**Tanggal Berlaku:** 7 Mei 2026

Di **SquishyFile**, privasi Anda adalah prioritas utama kami. Alat kami dirancang untuk memproses file secara langsung di browser web Anda (Client-side), memastikan data Anda tetap berada di bawah kendali Anda.

## 1. Pemrosesan File Lokal (Tanpa Unggah)
SquishyFile beroperasi pada arsitektur "Utamakan Privasi".
* **Tanpa Transfer Server:** File Anda **TIDAK PERNAH** diunggah, disimpan, atau diproses di server kami.
* **Eksekusi Lokal:** Semua tugas kompresi dan konversi terjadi sepenuhnya di perangkat lokal Anda.
* **Volatilitas Instan:** Setelah Anda menutup tab browser atau menghapus file, semua data segera dan secara permanen dihapus dari memori browser Anda.

## 2. Analitik & Pengumpulan Data
Kami menggunakan **Google Analytics** untuk memantau kinerja situs web dan meningkatkan pengalaman pengguna. Layanan ini mengumpulkan informasi non-pribadi dan anonim seperti:
* Jenis browser dan perangkat.
* Sumber rujukan dan halaman yang dikunjungi.
Data ini digunakan secara ketat untuk pengoptimalan teknis dan tidak menyertakan informasi apa pun dari file yang Anda proses.

## 3. Periklanan & Cookie
Kami menggunakan **Google AdSense** untuk menampilkan iklan, yang memungkinkan kami menjaga alat ini gratis untuk semua orang.
* Vendor pihak ketiga, termasuk Google, menggunakan cookie untuk menayangkan iklan berdasarkan kunjungan Anda sebelumnya ke situs web ini atau situs web lain.
* Penggunaan cookie periklanan oleh Google memungkinkannya dan mitranya untuk menayangkan iklan berdasarkan kunjungan Anda ke situs ini dan/atau situs lain di Internet.
* Anda dapat menyisih dari iklan hasil personalisasi kapan saja dengan mengunjungi [Setelan Iklan Google](https://www.google.com/settings/ads).

## 4. Tautan Pihak Ketiga
Situs web kami mungkin berisi tautan ke situs lain. Kami tidak bertanggung jawab atas praktik privasi atau konten situs web eksternal tersebut.

## 5. Perubahan pada Kebijakan ini
Kami berhak memperbarui Kebijakan Privasi ini kapan saja. Setiap perubahan akan tercermin pada halaman ini dengan "Tanggal Berlaku" yang diperbarui.
