# MOV、AVI、MKV、WebM 轉 MP4 有什麼差別？影片格式轉換指南

很多人搜尋影片格式轉換時，真正遇到的問題不是「副檔名不好看」，而是影片無法播放、無法上傳、無法剪輯，或傳給別人後對方打不開。這時候，把影片轉成 MP4 通常是最簡單的解法之一。

這篇文章整理常見影片格式的差異，以及 MOV、AVI、MKV、WebM、WMV、FLV、M4V、3GP、TS 等格式在什麼情況下適合轉成 MP4。

---

## 影片格式不只是副檔名

影片檔案通常包含兩個概念：

- 容器格式：例如 MP4、MOV、AVI、MKV、WebM
- 編碼格式：例如 H.264、HEVC、VP9、AV1、AAC 等

副檔名代表的是容器，但影片能不能播放，也和裡面的影像、音訊編碼有關。因此，有時候你看到的是 `.mp4`，但某些裝置仍然可能無法播放；也有些 `.mkv` 或 `.avi` 在特定播放器中可以正常開啟。

適合情境：

- iPhone 影片要跨平台分享
- Mac 匯出的 MOV 要上傳平台
- QuickTime 影片要放進剪輯軟體

## AVI 轉 MP4：適合舊影片與相機影片

AVI 是較早期常見的影片容器。很多舊 Windows 影片、相機錄影、監視器檔案或封存影片可能是 AVI。轉成 MP4 後，通常更容易在現代裝置上播放。

適合情境：

- 舊電腦留下的 AVI 無法播放
- 相機或錄影軟體輸出的 AVI 要分享
- 想整理封存影片格式

可以使用 [AVI 轉 MP4](/zh/avi-to-mp4/)。

## MKV 轉 MP4：適合高畫質影片，但要注意字幕與多音軌

MKV 很彈性，常見於高畫質影片、多音軌影片、字幕影片與封存內容。轉成 MP4 後相容性更好，但字幕軌、章節、多音軌等進階資訊不一定能完全以相同方式保留。

適合情境：

- MKV 在手機或電視上無法播放
- 想讓影片更容易分享或上傳
- 只需要主要影片與主要音訊

可以使用 [MKV 轉 MP4](/zh/mkv-to-mp4/)。

## WebM 轉 MP4：適合瀏覽器錄影與網頁影片

WebM 常見於網頁、瀏覽器錄影、螢幕錄影與線上工具輸出。若你要把 WebM 上傳到不支援的 App，或放進剪輯軟體，MP4 會更方便。

適合情境：

- 螢幕錄影輸出 WebM
- 線上會議錄影需要分享
- 網頁素材需要轉成通用影片

可以使用 [WebM 轉 MP4](/zh/webm-to-mp4/)。

## 其他格式：WMV、FLV、M4V、3GP、TS

除了以上格式，你也可能遇到：

- WMV：舊 Windows 影片格式
- FLV：早期網頁影片常見格式
- M4V：Apple 生態系中常見
- 3GP：舊手機影片格式
- TS、MTS、M2TS：相機、錄影機、串流或光碟相關格式

這些格式如果要分享、上傳或剪輯，通常也可以考慮轉成 MP4。若不是單一格式需求，可以使用 [影片轉檔工具](/zh/video-converter/) 選擇輸出格式。

## 轉成 MP4 會讓影片變小嗎？

不一定。格式轉換的主要目的通常是相容性，而不是壓縮。檔案大小可能變小、變大或差不多，取決於編碼方式、畫質設定、原始檔案與轉換方式。

如果你的目標是「讓影片容量變小」，應該使用影片壓縮工具；如果你的目標是「讓影片更好播放或上傳」，才是影片轉檔的主要場景。
