## MP4 檔案為什麼這麼大？

MP4 本身是一種容器格式，裡面包含影像和音訊的壓縮資料。影響 MP4 檔案大小的關鍵因素：

| 因素 | 說明 |
|------|------|
| **解析度** | 4K > 1080p > 720p，解析度越高越大 |
| **Bitrate（位元率）** | 每秒傳輸的資料量，越高畫質越好但檔案越大 |
| **影片長度** | 時間越長，自然越大 |
| **幀率（FPS）** | 60fps 比 30fps 大約大一倍 |
| **編碼格式** | H.265 比 H.264 更省空間 |

## 線上免費 MP4 壓縮怎麼做

### 步驟一：開啟壓縮工具

在瀏覽器打開線上 MP4 壓縮工具，不需要安裝任何程式，手機和電腦都能用。

### 步驟二：載入 MP4 檔案

點選「選擇檔案」，從你的裝置選取要壓縮的 MP4 影片。支援直接拖放到頁面上。

### 步驟三：選擇壓縮模式

一般工具提供幾種模式：

- **目標檔案大小**：設定你希望的最終大小（如 10MB、25MB）
- **壓縮比例**：設定要縮小到原本的幾成
- **畫質等級**：高/中/低品質，系統自動調整 bitrate

### 步驟四：開始壓縮並下載

點「開始壓縮」，等待幾秒至幾分鐘（視檔案大小而定），完成後下載壓縮後的 MP4 即可。

## MP4 壓縮的畫質與大小如何平衡？

很多人擔心壓縮後畫質變差，其實只要設定正確，差異非常小。

**實際測試參考：**

- 原始 1080p MP4，100MB → 壓縮後約 30～40MB，肉眼幾乎無差異
- 原始 4K MP4，500MB → 壓縮後約 80～150MB，在手機螢幕看幾乎相同

**建議設定：**
- 手機傳送用途：720p、Bitrate 1500～2000kbps
- 社群媒體上傳：1080p、Bitrate 2500～4000kbps
- 保存用途：盡量保留高品質，只調整不必要的部分
