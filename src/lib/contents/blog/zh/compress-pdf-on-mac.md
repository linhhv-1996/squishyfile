# Mac 用戶必看：如何在 Mac 上進行 PDF 壓縮？內建工具與免安裝神器推薦

當你需要透過 Email 傳送 PDF，或是上傳文件到某個平台時，「檔案太大」幾乎是最常見的困擾。許多 Mac 用戶第一時間會想到用內建的預覽程式來處理，但壓縮後的結果往往讓人失望——圖片變得模糊、文字出現鋸齒，甚至連原本清晰的圖表都糊成一團。

如果你也曾為了 pdf 壓縮 mac 的方法而苦惱，這篇文章會帶你了解預覽程式的運作原理、它的限制在哪裡，以及一個更聰明、不需要安裝任何軟體的替代方案。

## Mac 內建的「預覽程式」如何縮小 PDF 容量？

macOS 的預覽程式不只是一個看圖軟體，它其實內建了 PDF 的匯出與重新編碼功能。對於臨時需要減少檔案大小的情境，這確實是最快能上手的方式。

### 使用預覽程式壓縮的步驟

1. 用預覽程式開啟你想要處理的 PDF 檔案。
2. 點選上方選單列的「檔案」，選擇「輸出…」（注意：不是「輸出為 PDF」）。
3. 在彈出的視窗底部，找到「Quartz 濾鏡」下拉選單，選擇「Reduce File Size」。
4. 指定儲存位置與檔名，按下「儲存」。

完成這幾個步驟後，你會得到一個新的 PDF 檔案。從檔案資訊來看，容量確實變小了，但這個方法有一個幾乎所有 Mac 用戶都會遇到的致命缺點。

## 為什麼 Mac 內建壓縮會讓畫質變得極差？

預覽程式背後使用的是 macOS 的 Quartz 濾鏡系統。而 Apple 內建的「Reduce File Size」這個濾鏡，為了追求極致的檔案縮減效果，預設會將圖片品質壓到非常低。這意味著無論你原本的 PDF 有多清晰，經過這個濾鏡處理後，裡面的圖片都會被大幅降解析度，文字邊緣也可能因為過度壓縮而變得模糊。

對於內含照片、設計稿、建築藍圖或財務報表的文件來說，這種品質損失通常是不可接受的。你可能會發現，壓縮後的檔案雖然變小了，但已經不適合正式提交給客戶或主管。

### 想改善畫質？你可能需要修改系統設定檔

有些進階用戶會嘗試透過修改 macOS 系統中的 Quartz 濾鏡設定檔來調整預設品質。這些檔案通常位於系統深層目錄，需要手動編輯 XML 格式的參數，甚至還要處理系統保護機制（SIP）的問題。整個過程不僅繁瑣，而且一旦改錯設定，可能會影響到其他系統功能的運作。對於一般使用者來說，這條路實在太過曲折，也不符合現代人追求效率的工作流程。

## 常見問題 FAQ

**Q1: Mac 預覽程式壓縮後的 PDF 畫質很差，有辦法救回來嗎？**
一旦用「Reduce File Size」匯出，圖片資訊已經被永久降低解析度，原則上是無法還原的。建議保留原始檔案，並改用其他工具重新進行高畫質壓縮。

**Q2: 有沒有推薦的免安裝 pdf 壓縮 工具 可以在 Mac 上使用？**
Squishyfile 是目前相當推薦的選擇。它完全在瀏覽器內運作，不需要安裝任何軟體，也沒有作業系統版本限制，無論是 Intel 還是 Apple Silicon 的 Mac 都能順暢使用。

**Q3: 如何在 Mac 上 pdf 縮小容量，同時又不影響列印品質？**
關鍵在於壓縮演算法是否「聰明」。預覽程式的 Quartz 濾鏡會無差別降低所有內容的品質，而像 Squishyfile 這類工具則會針對不同內容類型進行最佳化，保留文字與線條的銳利度，只壓縮真正可以減肥的部分。

**Q4: 使用線上工具處理 PDF，我的資料會不會被洩漏？**
這取決於工具的運作方式。Squishyfile 採用 WASM 技術，檔案全程在你的裝置上處理，不會上傳到遠端伺服器，因此即使處理機密文件，也不用擔心資料外洩的風險。

**Q5: Apple Silicon Mac 壓縮 PDF 會比 Intel Mac 更快嗎？**
因為 Squishyfile 是在瀏覽器本地端運算，處理速度確實會受硬體效能影響。搭載 M 系列晶片的 Mac 在執行這類運算時，通常會比舊款 Intel 機型有更流暢的體驗。
