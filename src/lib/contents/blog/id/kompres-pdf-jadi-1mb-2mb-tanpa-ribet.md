# Cara Kompres PDF Jadi 1MB atau 2MB – Tanpa Ribet, Langsung Jadi

Banyak portal pendaftaran, sistem HR, dan formulir online yang membatasi ukuran file PDF yang bisa diunggah — biasanya di angka **1 MB atau 2 MB**. Kalau file kamu melebihi batas itu, unggahan akan gagal atau sistem menolak file kamu.

Kabar baiknya: ada cara cepat untuk kompres PDF sampai ukuran yang dibutuhkan, tanpa harus instal software atau paham teknis komputer.

---

## Kenapa Batas 1MB dan 2MB Begitu Umum?

Batas ukuran file ini bukan sembarangan. Ada beberapa alasan mengapa banyak sistem menetapkan batas ini:

- **Kapasitas server**: Server yang menerima dokumen dari ribuan pengguna butuh batasan agar tidak overload.
- **Kecepatan proses**: File lebih kecil lebih cepat diproses, diperiksa, dan diarsipkan oleh sistem.
- **Standar dokumen**: Dokumen teks standar seperti CV, ijazah, atau surat keterangan seharusnya tidak perlu lebih dari 1–2 MB.

Masalah terjadi saat kamu menghasilkan PDF dari scan fisik atau ekspor langsung dari aplikasi — hasilnya sering kali besar karena tidak dioptimalkan.

---

## Tips Agar Hasil Kompres Lebih Kecil

Jika file PDF kamu masih terlalu besar setelah dikompres, coba langkah-langkah berikut:

### 1. Pastikan PDF Tidak Berisi Halaman yang Tidak Perlu
Banyak orang lupa bahwa PDF mereka mengandung halaman kosong atau halaman yang tidak relevan. Hapus dulu halaman-halaman tersebut sebelum dikompres.

### 2. Kurangi Resolusi Gambar Sebelum Membuat PDF
Kalau kamu membuat PDF dari foto atau scan, pastikan resolusi gambar tidak berlebihan. Untuk dokumen standar, 150 DPI sudah cukup — tidak perlu 300 atau 600 DPI.

### 3. Gunakan Mode Hitam Putih untuk Scan Dokumen
Scan dokumen teks (KTP, ijazah, surat) dalam mode grayscale atau hitam putih menghasilkan file yang jauh lebih kecil dibanding mode warna penuh.

### 4. Simpan PDF dari Word/Excel dengan Benar
Kalau kamu ekspor dari Microsoft Word atau Google Docs, pilih opsi "Optimalkan untuk web" atau "Ukuran file minimum" saat menyimpan sebagai PDF.

## Berapa Besar Kompresi yang Bisa Dicapai?

Berikut gambaran umum hasil kompresi berdasarkan jenis file:

| Kondisi File Awal | Ukuran Awal | Estimasi Hasil |
|---|---|---|
| PDF dari scan HP | 5–10 MB | 0.5–2 MB |
| PDF dari Word/Excel | 3–5 MB | 300 KB–1 MB |
| PDF presentasi (banyak gambar) | 10–20 MB | 2–5 MB |
| PDF gabungan beberapa halaman scan | 8–15 MB | 1–3 MB |

*Hasil bervariasi tergantung konten. File yang sudah dikompres sebelumnya mungkin tidak bisa dikecilkan lebih lanjut secara signifikan.*

## Untuk Apa Biasanya Orang Kompres PDF ke 1MB atau 2MB?

Beberapa kebutuhan yang paling sering muncul:

- **Daftar CPNS atau seleksi ASN** — sistem SSCASN sering mensyaratkan dokumen di bawah 1 MB per file.
- **Upload di portal beasiswa** — LPDP, Beasiswa Unggulan, dan program lain biasanya ada batas ukuran ketat.
- **Lamaran kerja online** — banyak portal HRD dan sistem ATS yang membatasi ukuran CV dan portofolio.
- **Kirim via email** — lampiran email lebih cepat terkirim dan tidak ditolak server penerima.
- **Form pendaftaran kampus** — baik S1, S2, maupun program profesi.

---

## Apakah Kualitas Dokumen Tetap Terjaga?

Untuk dokumen yang berisi teks (CV, surat, formulir terisi), kompresi hampir tidak mengubah kualitas sama sekali. Teks tetap tajam dan bisa dibaca dengan jelas baik di layar maupun saat dicetak.

Untuk dokumen yang banyak mengandung foto atau gambar, mungkin ada sedikit penurunan kualitas gambar — tapi tetap dalam batas yang wajar dan tidak mengganggu keterbacaan.
