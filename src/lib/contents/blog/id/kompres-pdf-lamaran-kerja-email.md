# Kompres PDF untuk Lamaran Kerja & Email – Ukuran Tepat, Upload Berhasil

Sudah susah payah menyiapkan dokumen lamaran, eh malah gagal diupload karena ukuran file terlalu besar. Atau dokumen tidak bisa dikirim via email karena melewati batas lampiran.

Situasi ini sangat umum terjadi — dan solusinya sederhana: **kompres PDF kamu sebelum dikirim atau diupload.**

---

## Kenapa Dokumen Lamaran Harus Dikompres?

Hampir semua portal lowongan kerja dan sistem pendaftaran online memiliki batas ukuran file. Ini bukan peraturan yang dibuat-buat — ada alasan teknisnya:

- **Server kapasitas terbatas**: Sistem yang menerima ribuan lamaran per hari tidak bisa menyimpan file besar dari setiap pelamar.
- **Proses review lebih cepat**: File ringan lebih cepat dibuka oleh HRD, terutama jika mereka me-review dari HP.
- **Konsistensi sistem**: Batas ukuran memastikan semua pelamar menyerahkan dokumen dalam format yang wajar.

---

## Batas Ukuran File di Portal Umum Indonesia

Setiap platform punya ketentuan berbeda. Berikut gambaran umum batas ukuran yang sering ditemui:

| Platform / Sistem | Batas Ukuran per File |
|---|---|
| Portal SSCASN (CPNS) | 200 KB – 1 MB (tergantung jenis dokumen) |
| Portal job online umum | 2–5 MB |
| Email (Gmail, Yahoo, Outlook) | Lampiran total max 25 MB |
| WhatsApp | 100 MB per file |
| Portal beasiswa (LPDP, dll) | 1–2 MB |
| Portal kampus / universitas | 1–5 MB |
| Sistem HR perusahaan (ATS) | 2–10 MB |

Perhatikan bahwa untuk CPNS dan portal pemerintah, batasnya bisa sangat ketat — bahkan sampai 200 KB untuk beberapa jenis dokumen.

## Dokumen Apa Saja yang Perlu Dikompres?

Untuk keperluan lamaran kerja, dokumen yang paling sering perlu dikompres:

**CV / Resume**
CV yang dibuat dari template desain grafis atau PowerPoint sering berukuran besar (3–10 MB) karena banyak mengandung elemen visual. Padahal kontennya simpel.

**Ijazah dan Transkrip**
Scan ijazah atau transkrip nilai biasanya besar karena scanner menghasilkan gambar resolusi tinggi.

**KTP dan Dokumen Identitas**
Foto KTP yang discan atau difoto dengan HP beresolusi tinggi bisa mencapai 3–5 MB per file.

**Sertifikat dan Portofolio**
Sertifikat pelatihan, sertifikat bahasa, atau portofolio kerja bisa sangat besar terutama kalau banyak halaman atau mengandung gambar.

**Surat Lamaran**
Biasanya kecil kalau dibuat dari Word, tapi bisa besar kalau menggunakan template desain.

## Tips Khusus untuk Dokumen Lamaran

### ✅ Cek Persyaratan Terlebih Dahulu
Sebelum mengompres, baca dengan teliti persyaratan ukuran file di portal yang dituju. Beberapa portal menyebutkan batas dalam KB, bukan MB — jangan sampai salah tafsir.

### ✅ Simpan Satu Folder Khusus
Buat folder "Dokumen Lamaran – Sudah Kompres" di komputer atau HP kamu. Simpan semua versi yang sudah dikompres di sana agar tidak bingung menggunakan versi yang salah.

### ✅ Cek Kualitas Setelah Kompres
Buka file hasil kompresi dan pastikan semua teks masih bisa dibaca, gambar atau foto masih jelas, dan tidak ada halaman yang hilang.

### ✅ Beri Nama File yang Jelas
Namai file dengan format yang rapi, misalnya: `CV_NamaKamu_2026.pdf` atau `Ijazah_S1_NamaKamu.pdf`. Nama file yang profesional memberi kesan baik.

## Bagaimana dengan Dokumen Berpassword?

Beberapa orang menerima dokumen PDF berpassword dari institusi resmi (contoh: surat keterangan dari perusahaan sebelumnya, dokumen perbankan, dll). Jika kamu perlu mengirim dokumen seperti ini sebagai lampiran lamaran, kamu perlu menghapus passwordnya terlebih dahulu.

Tool kami menyediakan fitur **Remove Password** bersamaan dengan kompresi:

1. Upload file PDF berpassword.
2. Centang opsi **"Remove Password"**.
3. Masukkan password dokumen di kolom yang tersedia.
4. Klik **Kompres** — file akan dikompres dan password dihapus sekaligus.

Hasilnya adalah PDF bersih tanpa password yang siap dilampirkan ke lamaran.

## Kompres PDF untuk Kirim via Email

Untuk pengiriman via email, tips berikut membantu:

**Batas lampiran Gmail:** 25 MB total per email. Tapi praktisnya, lampiran besar sering lebih lambat terkirim dan lebih mungkin masuk folder spam.

**Rekomendasi ukuran ideal per file:**
- CV dan surat lamaran: di bawah 500 KB
- Dokumen pendukung (ijazah, sertifikat): di bawah 1 MB per file
- Total semua lampiran per email: di bawah 5 MB

Dengan mengompres file terlebih dahulu, email lebih cepat terkirim, lebih kecil kemungkinan masuk spam, dan lebih mudah dibuka penerima dari HP.

## Pertanyaan Umum

**Apakah kualitas ijazah atau sertifikat tetap terbaca setelah dikompres?**
Ya. Teks dalam dokumen tetap tajam dan jelas setelah kompresi. Untuk scan, kualitasnya tetap nyaman dibaca.

**Apakah data pribadi di dokumen saya aman?**
File diproses di browser kamu sendiri. Tidak ada data yang dikirim ke server kami.

**Kalau file sudah dikompres tapi masih terlalu besar, apa yang bisa dilakukan?**
Coba scan ulang dengan resolusi lebih rendah (150 DPI cukup), gunakan mode grayscale, atau pisahkan dokumen menjadi beberapa file jika portal memungkinkan.

**Apakah bisa kompres langsung dari HP saat mengisi formulir online?**
Bisa. Buka tool ini dari browser HP kamu, upload file, kompres, download, lalu lampirkan ke formulir.
