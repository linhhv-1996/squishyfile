# MacとWindowsでPDFを圧縮するには？標準機能とおすすめツール

PDFのファイルサイズが大きすぎて、メールに添付できなかったり、ストレージ容量を圧迫したりして困った経験はありませんか？

実は、MacやWindowsには、新しく専用のソフトウェアをインストールしなくても、OSに標準搭載されている機能を使ってPDFを圧縮する方法があります。本記事では、**「pdf 圧縮 mac」**や**「pdf 圧縮 windows」**と検索して手軽な解決策を探している方に向けて、それぞれの標準機能を使った手順を詳しく解説します。

ただし、OS標準の機能には「画質が著しく低下する」「手順が少し面倒で圧縮率が安定しない」といった弱点も存在します。そこで記事の後半では、Mac・Windowsを問わず、**画質を保ったまま手軽にPDFを圧縮できるおすすめのWebツール**も合わせてご紹介します。

## Macの標準機能「プレビュー」でPDFを圧縮する方法

Macユーザーの場合、標準アプリである「プレビュー」に内蔵されている「Quartzフィルタ」機能を利用することで、簡単にPDFの容量を減らすことができます。これは**pdf 圧縮 mac**での最も一般的で素早い解決策です。

### 圧縮の手順
1. 圧縮したいPDFファイルをダブルクリックし、**「プレビュー」**アプリで開きます。
2. 画面上部のメニューバーから**「ファイル」**をクリックし、**「書き出す…」**（「PDFとして書き出す」ではない点に注意）を選択します。
3. 表示された「書き出す」ダイアログボックスの下部にある**「Quartzフィルタ」**のドロップダウンメニューから、**「Reduce File Size（ファイルサイズを減らす）」**を選択します。
4. 保存先とファイル名を指定し、**「保存」**をクリックします。

> **注意点・デメリット：画質の劣化**
> この方法は非常に手軽な反面、ファイルサイズを強制的に小さくするため、**写真がぼやけたり、テキスト（文字）がにじんだりする**という大きなデメリットがあります。特に、細かな図表や高解像度の画像が含まれるビジネス書類、プレゼン資料などでは画質の低下が目立ちやすいため、用途に合わせて注意が必要です。

## Windowsの標準機能でPDFの容量を減らす方法

Windowsには、Macのプレビューのような「PDFのファイルサイズを減らす」という直接的な専用機能はありません。しかし、**「Microsoft Print to PDF」**という仮想プリンター機能を使って、PDFを再出力（仮想的に印刷）することで容量を軽量化できる場合があります。**pdf 圧縮 windows**でお悩みなら、まずはこの方法を試す価値があります。

### 圧縮の手順
1. 圧縮したいPDFファイルをブラウザ（Microsoft Edgeなど）やPDFリーダー（Adobe Acrobat Readerなど）で開きます。
2. メニューから**「印刷」**（またはショートカットキー `Ctrl` + `P`）を選択します。
3. プリンターの一覧（送信先）から**「Microsoft Print to PDF」**を選択します。
4. 印刷設定（用紙サイズやカラー/モノクロなど）を確認し、**「印刷」**ボタンをクリックします。
5. 新しいPDFファイルとして名前を付けて保存します。

> **注意点・デメリット：操作の煩雑さと不安定な圧縮率**
> 一度「印刷」というプロセスを挟むため、ページ数が多いファイルだと処理に時間がかかり、**操作がやや煩雑**に感じられます。また、新しいPDFが生成されることで軽くなるケースもありますが、圧縮率は安定しません。元のPDFの構成（画像が多いか等）によっては、**ファイルサイズがほとんど変わらない、あるいは逆に重くなってしまう**こともあり、作業効率の面で見逃せない弱点があります。
