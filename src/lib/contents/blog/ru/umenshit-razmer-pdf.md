# Как уменьшить размер PDF файла — онлайн и без программ

PDF занимает слишком много места или не проходит по лимиту для отправки? Уменьшить его размер можно прямо в браузере — бесплатно, без регистрации и без установки дополнительных программ.

Ниже — подробный разбор: почему файлы вырастают до нескольких десятков мегабайт, как их уменьшить и на что рассчитывать по результату.

---

## Что влияет на размер PDF файла

Размер PDF складывается из нескольких составляющих:

**Изображения** — самый весомый фактор. Встроенные фотографии и сканы без оптимизации могут занимать 90% объёма файла.

**Шрифты** — PDF встраивает шрифты целиком, даже если используется только часть символов. Несколько встроенных шрифтов — это лишние сотни килобайт.

**Метаданные** — информация о программе, авторе, истории правок. Обычно невелика, но в некоторых случаях добавляет объём.

**Версия PDF** — старые версии формата хранят данные менее эффективно.

**Дублирующиеся объекты** — программы иногда записывают одни и те же ресурсы несколько раз.

## Способы уменьшить размер PDF

### Способ 1: Онлайн-компрессор (самый быстрый)

Подходит для разовых задач. Не нужно ничего устанавливать.

1. Перейдите на онлайн-сервис сжатия PDF.
2. Загрузите файл — через кнопку или перетаскивание.
3. Нажмите **Сжать** или подождите автоматической обработки.
4. Скачайте результат.

Весь процесс занимает 1–2 минуты. Большинство сервисов поддерживают файлы до 100–500 МБ.

### Способ 2: Adobe Acrobat (если установлен)

Если у вас есть Adobe Acrobat (не Reader, а именно Acrobat):

1. Откройте PDF.
2. Выберите **Файл → Сохранить как другой → Уменьшить размер файла**.
3. Выберите целевую версию PDF (чем новее, тем лучше сжатие).
4. Сохраните.

Acrobat даёт лучший контроль над параметрами сжатия, но требует платной подписки.

### Способ 3: Через печать в PDF (Windows и macOS)

Бесплатный метод без дополнительного ПО:

**Windows:**
1. Откройте PDF в любом просмотрщике.
2. Нажмите **Печать**.
3. Выберите принтер **Microsoft Print to PDF**.
4. Сохраните новый файл.

**macOS:**
1. Откройте PDF в Просмотре (Preview).
2. **Файл → Экспорт как PDF**.
3. Нажмите **Фильтр кварца** → выберите **Уменьшить размер файла**.

Этот метод даёт умеренный результат — обычно 20–40% уменьшение.

### Способ 4: Ghostscript (для опытных пользователей)

Бесплатная утилита с гибкими настройками:

```bash
gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 \
   -dPDFSETTINGS=/ebook \
   -dNOPAUSE -dBATCH \
   -sOutputFile=output.pdf input.pdf
```

Параметр `-dPDFSETTINGS` определяет степень сжатия:
- `/screen` — максимальное сжатие, 72 DPI, только для просмотра на экране
- `/ebook` — баланс качества и размера, 150 DPI (рекомендуется)
- `/printer` — 300 DPI, для печати
- `/prepress` — 300 DPI, максимальное качество

## Реалистичные результаты уменьшения

Разные типы PDF сжимаются по-разному:

| Тип файла | Исходный размер | После сжатия | Экономия |
|---|---|---|---|
| Скан (1 стр.) | 4 МБ | 400–700 КБ | 80–90% |
| Скан (10 стр.) | 40 МБ | 3–6 МБ | 85–92% |
| Текстовый документ | 800 КБ | 400–600 КБ | 25–50% |
| Презентация | 20 МБ | 2–4 МБ | 80–90% |
| Технический чертёж | 5 МБ | 2–3 МБ | 40–60% |

**Важно:** Текстовые PDF сжимаются значительно хуже, чем файлы с изображениями. Если документ состоит только из текста, не ожидайте уменьшения в 10 раз.

## Уменьшить размер PDF для конкретных задач

### Для отправки по электронной почте

Большинство почтовых сервисов ограничивают вложения:
- Gmail — до 25 МБ
- Outlook — до 20 МБ
- Mail.ru — до 25 МБ

Сожмите PDF до 10–15 МБ с запасом. Если нужно отправить несколько файлов — сжимайте каждый отдельно или архивируйте в ZIP.

### Для загрузки на Госуслуги и государственные порталы

Стандартный лимит — 2–10 МБ на файл. Для сканов документов (паспорт, СНИЛС, диплом) рекомендуется сжимать до 500 КБ — 1 МБ.

### Для отправки в WhatsApp или Telegram

WhatsApp ограничивает файлы до 100 МБ, Telegram — до 2 ГБ (для обычных аккаунтов). Для мессенджеров жёстких ограничений обычно нет, но меньший файл загружается быстрее.

### Для хранения в облаке

Если место в облачном хранилище заканчивается — сжатие архивов PDF может освободить значительный объём. Сканы уменьшаются особенно сильно.

## Как уменьшить размер PDF на телефоне

На Android и iPhone метод тот же — онлайн-компрессор в браузере.

**Android:**
1. Откройте Chrome или другой браузер.
2. Перейдите на сервис сжатия PDF.
3. Нажмите «Загрузить файл» — откроется файловый менеджер.
4. Найдите PDF, выберите его.
5. Дождитесь обработки, скачайте результат.

**iPhone:**
1. Откройте Safari.
2. Перейдите на онлайн-компрессор.
3. Tap «Выбрать файл» — откроется приложение «Файлы».
4. Выберите PDF, дождитесь сжатия.
5. Нажмите «Скачать» — файл сохранится в «Файлы» → «Загрузки».

## Частые вопросы

**Можно ли уменьшить PDF без потери качества текста?**
Да. Текст в PDF — это векторные данные, которые не теряют качества при сжатии. Сжатие влияет только на изображения внутри файла.

**Почему PDF не уменьшается после сжатия?**
Возможные причины: файл уже оптимизирован; он состоит только из текста и шрифтов (резерв невелик); файл создан в формате PDF/A, который ограничивает сжатие.

**Можно ли уменьшить размер сканированного PDF?**
Да, и очень значительно. Сканы хранятся как изображения с высоким DPI — при сжатии разрешение уменьшается до комфортного для чтения, файл становится в 5–10 раз меньше.

**Безопасно ли загружать документы на онлайн-сервисы?**
Для обычных документов — да. Для конфиденциальных данных выбирайте сервисы с локальной обработкой в браузере или используйте офлайн-инструменты.

## Итог

Уменьшить размер PDF файла проще всего через онлайн-компрессор — это займёт меньше минуты. Для регулярных задач рассмотрите Ghostscript или Adobe Acrobat с возможностью тонкой настройки сжатия.

Сканированные документы уменьшаются сильнее всего — рассчитывайте на 80–90% экономии объёма при сохранении читаемости.
