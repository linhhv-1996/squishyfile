# Как сжать видео — онлайн и без программ

Видеофайл не отправляется в мессенджер, не загружается на сайт или занимает слишком много места? Это решаемо. Сжать видео можно онлайн — в браузере, без установки дополнительных программ.

Разберём все способы: от быстрого онлайн-сжатия до профессиональных инструментов для точного контроля.

---

## Почему видео такое большое

Видеофайл содержит тысячи кадров. Каждый кадр — это изображение. Без сжатия 1 минута видео 1080p весила бы около 10 ГБ.

Реальный размер зависит от:

- **Разрешения** — 4K в 4 раза «тяжелее» Full HD при прочих равных
- **Частоты кадров** — 60 fps вдвое больше, чем 30 fps
- **Кодека** — H.264 vs H.265 (HEVC): одинаковое качество, файл вдвое меньше
- **Битрейта** — количество данных в секунду: выше = лучше качество = больше файл
- **Длины видео** — очевидно, но часто забывают

Смартфоны снимают в высоком качестве «по умолчанию» — 4K 60fps на iPhone или Android-флагмане. 1 минута такого видео весит 350–500 МБ.

## Как сжать видео онлайн — пошаговая инструкция

**Шаг 1.** Откройте онлайн-компрессор видео в браузере.

**Шаг 2.** Загрузите видео — нажмите кнопку или перетащите файл. Поддерживаются форматы MP4, MOV, AVI, MKV и другие.

**Шаг 3.** Выберите параметры сжатия, если сервис их предлагает: качество, разрешение, формат.

**Шаг 4.** Нажмите **Сжать** или **Конвертировать**.

**Шаг 5.** Дождитесь обработки — время зависит от размера файла и скорости интернета. Обычно 1–5 минут для файла до 500 МБ.

**Шаг 6.** Скачайте сжатое видео.

**Важно:** большинство бесплатных онлайн-сервисов ограничивают размер файла (обычно 500 МБ – 2 ГБ). Для больших файлов нужна программа.

## Сколько весит видео после сжатия

Реалистичные ожидания:

| Исходный файл | Разрешение | До сжатия | После сжатия |
|---|---|---|---|
| 1 мин. с телефона (4K) | 3840×2160 | 400–500 МБ | 50–100 МБ |
| 1 мин. с телефона (1080p) | 1920×1080 | 100–150 МБ | 20–40 МБ |
| 5 мин. запись экрана | 1920×1080 | 300–500 МБ | 30–80 МБ |
| 30 сек. для соцсетей | 1080×1080 | 50–80 МБ | 5–15 МБ |
| Видеозвонок (720p) | 1280×720 | 80–120 МБ | 15–30 МБ |

## Способы сжатия видео

### Онлайн-компрессор (самый простой)

Подходит для разовых задач. Не нужно ничего устанавливать. Ограничение — размер файла и скорость интернета.

### HandBrake (бесплатная программа, все платформы)

HandBrake — лучший бесплатный инструмент для сжатия видео на компьютере. Работает на Windows, macOS и Linux.

**Основные параметры:**
- **Пресет:** Web → Gmail Large 3 минуты 720p — хорошая отправная точка
- **Кодек:** H.265 (HEVC) — такое же качество при вдвое меньшем файле
- **Quality (RF):** 22–28. Чем выше число, тем больше сжатие и меньше качество. RF 22 = отличное качество, RF 28 = заметное снижение.

Скачать: handbrake.fr (бесплатно, без рекламы)

### ffmpeg (командная строка)

Для тех, кто не боится терминала — самый мощный и гибкий инструмент:

```bash
# Сжать видео с сохранением хорошего качества
ffmpeg -i input.mp4 -vcodec libx265 -crf 28 output.mp4

# Уменьшить разрешение до 720p
ffmpeg -i input.mp4 -vf scale=1280:720 -vcodec libx264 -crf 23 output.mp4

# Задать целевой битрейт
ffmpeg -i input.mp4 -b:v 1M output.mp4
```

Параметр `-crf`: 18 = почти без потерь, 23 = хорошее качество (стандарт), 28 = заметное сжатие, 35+ = низкое качество.

## Сжать видео для конкретных платформ

Каждая платформа имеет свои ограничения и рекомендации:

**Telegram**
- Максимальный размер: 2 ГБ (обычный аккаунт), 4 ГБ (Premium)
- Рекомендуется: MP4, H.264, до 1080p

**WhatsApp**
- Максимальный размер: 100 МБ
- Рекомендуется: MP4, 720p, до 3 минут длиной

**Instagram / Reels**
- Максимальный размер: 100 МБ (для Reels до 90 секунд)
- Рекомендуется: MP4, H.264, 1080×1920 (вертикальное), 30 fps

**YouTube**
- Максимальный размер: 128 ГБ или 12 часов
- YouTube перекодирует загруженное видео, поэтому особых ограничений нет

**Государственные порталы и системы документооборота**
- Обычно до 50–100 МБ, форматы MP4 или AVI
- Рекомендуется: 720p, H.264

## Частые вопросы

**Можно ли сжать видео без потери качества?**
Не полностью — видео использует сжатие с потерями (lossy). Но при умеренных настройках разницу на экране телефона или ноутбука не заметить. Переход с H.264 на H.265 даёт файл вдвое меньше при том же воспринимаемом качестве.

**Почему видео занимает столько места после съёмки на телефоне?**
Смартфоны снимают в высоком качестве, чтобы сохранить возможность редактирования. При съёмке 4K 60fps телефон не сжимает агрессивно — это делаете вы при финальной обработке.

**Какой формат лучше для сжатого видео?**
MP4 с кодеком H.264 — универсальный выбор: поддерживается везде. H.265 (HEVC) — лучшее качество при меньшем размере, но не везде воспроизводится.

**Как сжать видео на телефоне?**
Используйте онлайн-компрессор в браузере (см. отдельные статьи для Android и iPhone) или приложения для редактирования видео с функцией экспорта в меньшем размере.

## Итог

Сжать видео проще всего через онлайн-компрессор — подходит для файлов до 500 МБ и разовых задач. Для больших файлов и регулярного использования установите HandBrake: бесплатно, мощно, без ограничений по размеру.

Ключевые параметры: кодек H.265 вместо H.264 (вдвое меньший файл) и разрешение 1080p вместо 4K (в 4 раза меньше данных).
