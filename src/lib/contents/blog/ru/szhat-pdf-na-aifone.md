# Как сжать PDF на iPhone — без приложений, через браузер

PDF на iPhone стал слишком большим, чтобы отправить или загрузить? Его можно сжать прямо в Safari — без установки дополнительных приложений.

Метод работает на любом iPhone с iOS 13 и новее. Всё, что нужно — браузер и файл в приложении «Файлы».

---

## Почему PDF на iPhone бывают большими

iPhone создаёт крупные PDF в нескольких ситуациях:

**Сканирование в приложении «Заметки»** — Notes сканирует в высоком разрешении (300–400 DPI). Страница A4 весит 2–5 МБ. 5 страниц — уже 10–25 МБ.

**Экспорт из Pages или Keynote** — Apple-приложения встраивают изображения в максимальном качестве.

**Пересланные вложения** — PDF, присланные по email или через мессенджеры, часто уже несут лишний вес от исходного приложения.

**Скачанные с сайтов** — качество варьируется от оптимизированных до огромных несжатых файлов.

---

## Как сжать PDF на iPhone через Safari

Этот способ не требует скачивания ничего.

**Шаг 1.** Откройте **Safari** на iPhone.

**Шаг 2.** Перейдите на онлайн-компрессор PDF (поищите в Google «сжать PDF онлайн» прямо в Safari).

**Шаг 3.** Нажмите кнопку **Выбрать файл** или **Загрузить PDF** на странице сервиса.

**Шаг 4.** В появившемся меню выберите **«Файлы»** — откроется приложение «Файлы».

**Шаг 5.** Найдите нужный PDF и tap по нему.

**Шаг 6.** Дождитесь загрузки и обработки — обычно 10–30 секунд.

**Шаг 7.** Нажмите **Скачать** на странице сервиса.

**Шаг 8.** Сжатый PDF сохранится в **Файлы → Загрузки**.

## Где найти PDF на iPhone

Если не можете найти файл — вот где он может быть:

| Источник PDF | Где искать |
|---|---|
| Скан из Заметок | Файлы → iCloud Drive |
| Вложение из Mail | Mail → вложение → «Поделиться» → «Сохранить в Файлах» |
| Скачанный в Safari | Файлы → Загрузки |
| Из Pages / Keynote | Файлы → На iPhone → Pages / Keynote |
| Из WhatsApp / Telegram | Файлы → На iPhone → WhatsApp |

**Если файл не в «Файлах»:** откройте приложение, где лежит документ, нажмите «Поделиться» → «Сохранить в Файлах», затем используйте браузерный метод.

## Что ожидать по результату

Сжатие PDF на iPhone даёт такие же результаты, как и на компьютере:

| Тип PDF | До сжатия | После сжатия |
|---|---|---|
| Скан из Заметок (1 стр.) | 3–6 МБ | 300–700 КБ |
| Скан из Заметок (5 стр.) | 12–25 МБ | 1–3 МБ |
| Резюме с фото | 1–2 МБ | 200–500 КБ |
| Документ из Pages | 2–5 МБ | 500 КБ – 1 МБ |

Сканы из приложения «Заметки» сжимаются особенно хорошо — Notes создаёт изображения в разрешении, намного превышающем необходимое для чтения и отправки.

## Возможные проблемы и их решение

**Кнопка «Выбрать файл» открывает камеру, а не Файлы**
Это нормально — iOS показывает несколько источников. Нажмите на иконку «Просмотр» или «Файлы» в нижней части панели.

**После tap «Скачать» ничего не происходит**
Нажмите и удерживайте ссылку скачивания, выберите **«Скачать связанный файл»** — файл сохранится в «Загрузки».

**Safari блокирует скачивание**
Зайдите в **Настройки → Safari → Загрузки** и убедитесь, что выбрано место сохранения (iCloud Drive или «На iPhone»).

**Сервис показывает ошибку при загрузке**
Файл может быть защищён паролем. Снимите защиту через Adobe Acrobat или другой инструмент, затем попробуйте снова.

**Скачанный файл не появляется в Файлах**
Откройте Safari → нажмите на иконку загрузок (стрелка вниз в правом верхнем углу) → найдите файл там и сохраните вручную.

## Поделиться сжатым PDF прямо с iPhone

После скачивания сжатый файл можно сразу отправить:

1. Откройте **Файлы** → **Загрузки**.
2. Нажмите и удерживайте файл → **Поделиться**.
3. Выберите способ: **Mail**, **WhatsApp**, **Telegram**, **AirDrop** или любое другое приложение.

Или прямо из браузера: на многих сервисах после сжатия есть кнопка «Поделиться» — она открывает стандартное iOS-меню расшаривания.

## Приложения для сжатия PDF на iPhone (если нужны регулярно)

Если вы часто сжимаете PDF и хотите отдельное приложение:

**PDF Expert (Readdle)** — мощный редактор с функцией сжатия. Бесплатная базовая версия, расширенные функции по подписке.

**Scanner Pro (Readdle)** — сканирование с оптимизированным сохранением. Хорошо подходит, если PDF создаются на самом iPhone.

**Adobe Acrobat Reader** — бесплатный просмотрщик, сжатие требует подписки Acrobat.

Для большинства пользователей браузерный метод удобнее — не занимает место на устройстве и работает сразу.

## Часто задаваемые вопросы

**Можно ли сжать PDF на iPhone без интернета?**
Браузерные сервисы требуют подключение. Для офлайн-сжатия нужно специальное приложение, например PDF Expert.

**Сохраняется ли качество после сжатия на iPhone?**
Да. Текст остаётся чётким, структура документа не меняется. Незначительное снижение резкости возможно только у встроенных изображений.

**Можно ли сжать PDF прямо в приложении «Файлы»?**
Нет, iOS не имеет встроенной функции сжатия PDF в приложении «Файлы».

**Почему PDF, сканированный на iPhone, такой большой?**
Приложение «Заметки» сохраняет сканы в высоком разрешении для максимального качества. Это полезно для архивного хранения, но избыточно для отправки — именно поэтому сжатие так эффективно.

## Итог

Сжать PDF на iPhone проще всего через браузер Safari — без скачивания приложений. Откройте онлайн-компрессор, загрузите файл из «Файлов», скачайте результат. Весь процесс занимает меньше минуты.

Для сканов из приложения «Заметки» результат будет особенно заметным — такие файлы уменьшаются в 5–10 раз.
