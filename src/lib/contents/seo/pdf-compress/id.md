# Cara Kompres PDF

Tanpa akun, tanpa unggahan ke server — semua pemrosesan berlangsung di browser Anda. File Anda tidak pernah meninggalkan perangkat Anda.

## Langkah 1: Pilih File PDF

Seret dan lepas file PDF ke area unggah, atau klik untuk memilih file. Anda juga bisa mengklik **Telusuri** untuk membuka pemilih file. Semua jenis PDF didukung — dokumen hasil scan, presentasi, formulir, dan lainnya.

## Langkah 2: Pilih Mode Kompresi

Pilih mode yang sesuai dengan kebutuhan Anda:

- **Seimbang** — Mengurangi ukuran file secara signifikan sambil menjaga kejernihan gambar dan teks. Terbaik untuk berbagi sehari-hari, lampiran email, atau mengunggah ke formulir web.
- **Maksimum** — Mengompres file sekecil mungkin. Ideal saat ukuran file adalah prioritas utama, seperti untuk aplikasi pesan atau batas unggah yang ketat.

## Langkah 3: (Opsional) Hapus Proteksi Kata Sandi

Jika PDF Anda dilindungi kata sandi dan ingin membukanya, centang **Hapus proteksi kata sandi** lalu masukkan kata sandi saat ini. File yang dihasilkan adalah PDF yang bersih tanpa proteksi.

## Langkah 4: Mulai Kompresi

Klik **Kompres PDF**. Seluruh pemrosesan berjalan secara lokal di browser Anda menggunakan WebAssembly — tidak ada data yang dikirim ke server.

## Langkah 5: Unduh File Anda

Setelah kompresi selesai, Anda akan melihat ukuran asli, ukuran terkompresi, dan berapa banyak ruang yang dihemat. Klik **Unduh** untuk menyimpan PDF yang telah dioptimalkan ke perangkat Anda.

---

## Tips untuk Hasil Terbaik

- **Sudah dikompresi?** Jika PDF sebelumnya sudah dioptimalkan atau hanya berisi teks biasa, pengurangan ukuran mungkin tidak terlalu besar — ini adalah hal yang wajar.
- **PDF hasil scan** mengandung gambar tertanam dan biasanya dapat dikompresi dengan baik.
- **Presentasi besar** dengan banyak slide beresolusi tinggi paling diuntungkan dengan mode Maksimum.
- File Anda tidak pernah diunggah. Semua pemrosesan dilakukan sepenuhnya di browser, sehingga dokumen Anda tetap privat.
