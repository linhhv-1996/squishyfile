# PDF Compressor: Reduce PDF File Size Online for Free

Squishyfile is a free PDF compressor that helps you reduce PDF file size directly in your browser. No account, no software installation, and no complicated settings are required. Select a PDF, choose a compression mode, and download a smaller file that is easier to email, upload to a form, share with someone else, or store in the cloud.

Unlike many online PDF compression tools, Squishyfile processes your PDF in the browser using WebAssembly. Your PDF is not uploaded to a server for compression. The file is handled on your own device, which makes the tool useful for everyday documents that may contain personal information, such as resumes, invoices, contracts, forms, applications, and scanned paperwork.

## How to Compress a PDF

### Step 1: Select Your PDF

Drag and drop your PDF into the upload area, or click the drop zone to browse files on your device. You can use scanned documents, resumes, forms, reports, presentations, image-based PDFs, and many other standard PDF files.

If you are using a phone, choose the PDF from your Files app, Downloads folder, cloud storage app, or any location supported by your browser's file picker.

### Step 2: Choose a Compression Mode

Select the mode that matches your goal.

- **Balanced**: Best for most PDFs. It reduces file size while keeping text and images easy to read.
- **Maximum**: Best when the smallest possible file size matters more than image sharpness.
- **Remove password protection**: Use this option when your PDF is password-protected and you know the current password.

For everyday use, start with **Balanced** mode. It is usually the best choice for email attachments, resumes, business documents, and web form uploads. Use **Maximum** mode when you need to meet a strict file size limit, such as 1 MB, 2 MB, or 5 MB.

### Step 3: Enter a Password If Needed

If your PDF is protected with a password, check the password removal option and enter the current password. Squishyfile can unlock and compress the PDF in the same process, as long as you have permission to remove the password and know the correct password.

The downloaded file will be a standard PDF without the old password protection. For more details, read the guide on [how to remove a password from a PDF](/blog/remove-password-from-pdf-free-online).

### Step 4: Compress the PDF

Click **Compress PDF** to start. The compression runs in your browser, so the file is not sent to a remote server. Processing time depends on the number of pages, the amount of image data, the original file size, and the speed of your device.

Small PDFs often finish in a few seconds. Large scanned documents or image-heavy presentations may take longer, especially on older phones or low-memory devices.

### Step 5: Download the Compressed PDF

After compression is complete, you will see the original file size, the compressed file size, and the amount of space saved. Click **Download** to save the smaller PDF to your device.

Before sending an important file, open the compressed PDF and quickly check that every page looks correct. Text should remain readable, images should still be clear enough for your purpose, and the page order should stay the same.

## When the PDF File Size Does Not Decrease Much

Sometimes a PDF will only shrink slightly. This usually happens for one of these reasons:

- The PDF was already compressed before
- The file contains mostly text and vector graphics
- There are very few images to optimize
- The document contains fonts, forms, or metadata that cannot be reduced much
- The PDF has already been exported with efficient settings

If the compressed file is still too large, try these steps:

- Use **Maximum** mode instead of **Balanced** mode
- Remove blank, duplicate, or unnecessary pages before compressing
- Recreate scanned pages at a lower scan resolution if possible
- Avoid compressing the same PDF repeatedly, because repeated lossy compression can reduce image quality
- Check the output at 100% zoom before submitting or sharing it

## Compress PDF to 1 MB or Another Size Limit

Many upload portals reject PDFs that are larger than a fixed limit. Job applications, school forms, government portals, email systems, and internal company tools may require a PDF under 1 MB, 2 MB, 5 MB, or 10 MB.

To target a specific size, start with **Balanced** mode. If the result is still too large, try **Maximum** mode. If the file remains above the limit, the PDF may contain too much content to fit under that size without removing pages or reducing image quality further.

For example, a scanned document with only a few pages may compress below 1 MB easily. A long photo-heavy portfolio may need more aggressive optimization or fewer pages. For more practical tips, see [compress PDF to 1 MB or less](/blog/compress-pdf-to-1mb-online-free).

## Are PDFs Uploaded to a Server?

No. Squishyfile compresses PDFs locally in your browser using WebAssembly. Your file is not uploaded to a server for processing.

This is different from many online PDF tools that send your file to a remote server, process it there, and then send the result back. Browser-based processing helps reduce privacy risk because the PDF stays on your device during compression.

This matters when working with sensitive documents such as contracts, tax forms, financial statements, medical records, identity documents, and private business files. You should still store the downloaded PDF carefully, especially if you removed password protection from the original file.
