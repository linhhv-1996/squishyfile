# Cómo comprimir un vídeo

Sin cuenta, sin subida a servidores — todo el procesamiento ocurre en tu navegador. Tu vídeo nunca sale de tu dispositivo.

## Paso 1: Selecciona tu vídeo

Arrastra y suelta el archivo de vídeo en la zona de carga, o haz clic en **Examinar** para abrirlo desde el explorador. Se admiten MP4, MOV, MKV, AVI, WebM y otros formatos habituales.

## Paso 2: Elige el método de compresión

Hay dos formas de controlar el tamaño de salida:

### Preajustes rápidos
Selecciona el preajuste según tus necesidades:

- **Bajo** — El tamaño de archivo más pequeño posible. Algo de pérdida de calidad, pero una reducción drástica del peso. Ideal para apps de mensajería o archivar grabaciones antiguas.
- **Equilibrado** — El punto óptimo entre tamaño y calidad. Recomendado para la mayoría de usos: redes sociales, correo o compartir en general.
- **Alto** — Pérdida de calidad mínima con compresión moderada. Mejor cuando la calidad visual importa, como en presentaciones o entregas a clientes.

### Tamaño objetivo
¿Necesitas cumplir un límite concreto? Introduce el tamaño en MB o pulsa uno de los botones de selección rápida para plataformas habituales:

| Plataforma | Límite |
|---|---|
| LINE | 10 MB |
| Discord | 8 MB |
| Gmail | 25 MB |
| Messenger | 20 MB |
| Telegram | 50 MB |
| Twitter/X | 16 MB |
| Correo electrónico | 100 MB |

Cuando se establece un tamaño objetivo, los preajustes se desactivan y el compresor ajusta automáticamente la calidad para alcanzar tu límite.

## Paso 3: Comprimir

Haz clic en **Comprimir ahora**. Una barra de progreso muestra el avance. La compresión se ejecuta completamente en tu navegador mediante FFmpeg vía WebAssembly — el vídeo nunca se envía a ningún lugar.

> **Mantén la pestaña abierta** mientras dure la compresión. Cambiar de pestaña o minimizar el navegador puede interrumpir el proceso.

## Paso 4: Descarga

Al terminar, verás el tamaño original, el comprimido y el espacio ahorrado. Haz clic en **Descargar** para guardar el MP4 comprimido en tu dispositivo.

---

## Consejos para mejores resultados

- **Los clips cortos se comprimen más rápido.** Los vídeos muy largos (30 min+) pueden tardar varios minutos según el dispositivo.
- **¿Ya estaba comprimido?** Si un vídeo fue codificado con bajo bitrate, una compresión adicional puede no reducir mucho el tamaño.
- **El tamaño objetivo es una estimación.** El tamaño real puede variar ligeramente según el contenido y la complejidad del vídeo.
- **Las grabaciones de pantalla y las animaciones** se comprimen muy bien — a menudo entre un 70 y un 90%.
- Tus archivos nunca se suben a ningún servidor. Todo el procesamiento ocurre en el dispositivo, tus vídeos permanecen completamente privados.
