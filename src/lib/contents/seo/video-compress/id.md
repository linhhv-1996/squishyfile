# Cara Kompres Video

Tanpa akun, tanpa unggahan ke server — semua pemrosesan berlangsung di browser Anda. Video tidak pernah meninggalkan perangkat Anda.

## Langkah 1: Pilih File Video

Seret dan lepas file video ke area unggah, atau klik **Telusuri** untuk membuka pemilih file. Format MP4, MOV, MKV, AVI, WebM, dan format umum lainnya didukung.

## Langkah 2: Pilih Metode Kompresi

Ada dua cara untuk mengontrol ukuran output:

### Preset Cepat
Pilih preset sesuai kebutuhan Anda:

- **Rendah** — Ukuran file sekecil mungkin. Ada sedikit penurunan kualitas, tapi ukuran berkurang drastis. Cocok untuk aplikasi pesan atau mengarsipkan rekaman lama.
- **Seimbang** — Keseimbangan terbaik antara ukuran dan kualitas. Direkomendasikan untuk kebanyakan kasus: media sosial, email, atau berbagi umum.
- **Tinggi** — Penurunan kualitas minimal dengan kompresi sedang. Terbaik saat kualitas visual penting, seperti presentasi atau pekerjaan klien.

### Target Ukuran File
Perlu memenuhi batas tertentu? Masukkan ukuran target dalam MB, atau ketuk salah satu tombol cepat untuk platform umum:

| Platform | Batas |
|---|---|
| LINE | 10 MB |
| Discord | 8 MB |
| Gmail | 25 MB |
| Messenger | 20 MB |
| Telegram | 50 MB |
| Twitter/X | 16 MB |
| Email | 100 MB |

Ketika target ukuran diatur, preset dinonaktifkan — kompressor secara otomatis menyesuaikan kualitas untuk memenuhi batas Anda.

## Langkah 3: Mulai Kompresi

Klik **Kompres Sekarang**. Progress bar menunjukkan kemajuan proses. Kompresi berjalan sepenuhnya di browser Anda menggunakan FFmpeg via WebAssembly — video tidak pernah dikirim ke mana pun.

> **Biarkan tab tetap terbuka** selama kompresi berlangsung. Beralih tab atau meminimalkan browser dapat menghentikan proses.

## Langkah 4: Unduh

Setelah selesai, Anda akan melihat ukuran asli, ukuran terkompresi, dan ruang yang dihemat. Klik **Unduh** untuk menyimpan MP4 terkompresi ke perangkat Anda.

---

## Tips untuk Hasil Terbaik

- **Klip pendek dikompres paling cepat.** Video sangat panjang (30 menit+) mungkin membutuhkan beberapa menit tergantung perangkat Anda.
- **Sudah dikompresi?** Jika video sebelumnya sudah dikodekan dengan bitrate rendah, kompresi lebih lanjut mungkin tidak banyak mengurangi ukuran.
- **Target ukuran adalah perkiraan.** Ukuran output aktual mungkin sedikit berbeda tergantung konten dan kompleksitas video.
- **Rekaman layar dan animasi** sangat mudah dikompres — sering kali berkurang 70–90%.
- File Anda tidak pernah diunggah. Semua pemrosesan dilakukan di perangkat, video Anda tetap sepenuhnya privat.
