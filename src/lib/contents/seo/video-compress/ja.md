# 動画を圧縮する方法

アカウント不要、サーバーへのアップロード不要。すべての処理はブラウザ上で完結します。

## ステップ 1：動画を選択する

動画ファイルをドロップゾーンにドラッグ＆ドロップするか、**参照**ボタンからファイルを選択してください。MP4・MOV・MKV・AVI・WebM など、主要なフォーマットに対応しています。

## ステップ 2：圧縮方法を選ぶ

出力サイズの調整方法は2通りあります。

### クイックプリセット
目的に合わせてプリセットを選択してください：

- **低画質（最小サイズ）** — ファイルを最小限まで圧縮します。多少の画質低下はありますが、容量を大幅に削減できます。メッセージアプリでの送信や古い動画のアーカイブに最適です。
- **バランス** — サイズと画質のバランスが取れた設定。SNSへの投稿やメール送信、一般的な共有など、ほとんどの用途におすすめです。
- **高画質** — 画質をできる限り保ちながら適度に圧縮します。プレゼン資料やクライアントへの納品物など、画質が重要な場合に最適です。

### ターゲットファイルサイズ
特定のサイズ制限に合わせたい場合は、目標サイズをMBで直接入力するか、よく使われるプラットフォームのボタンをタップしてください：

| プラットフォーム | 上限 |
|---|---|
| LINE | 10 MB |
| Discord | 8 MB |
| Gmail | 25 MB |
| Messenger | 20 MB |
| Telegram | 50 MB |
| Twitter/X | 16 MB |
| メール | 100 MB |

ターゲットサイズを設定すると、プリセットは無効になり、コンプレッサーが自動的に画質を調整して目標サイズに近づけます。

## ステップ 3：圧縮を実行する

**今すぐ圧縮**ボタンをクリックしてください。プログレスバーで進捗を確認できます。圧縮処理はWebAssembly経由のFFmpegを使用してブラウザ内でローカルに実行されます。動画がデバイス外に送信されることは一切ありません。

> **圧縮中はタブを開いたままにしてください。** タブを切り替えたりブラウザを最小化すると、処理が一時停止または中断する場合があります。

## ステップ 4：ダウンロードする

完了後、元のサイズ・圧縮後のサイズ・削減量が表示されます。**ダウンロード**をクリックして、圧縮されたMP4をデバイスに保存してください。

---

## より良い結果を得るためのヒント

- **短い動画ほど速く圧縮できます。** 非常に長い動画（30分以上）はデバイスによっては数分かかる場合があります。
- **すでに圧縮済みの動画の場合：** 低ビットレートでエンコードされた動画は、さらに圧縮してもサイズが大きく変わらないことがあります。
- **ターゲットサイズはあくまで目安です。** 動画の内容や複雑さによって、実際の出力サイズが若干異なる場合があります。
- **スクリーン録画やアニメーション**は特に圧縮効果が高く、70〜90%の削減が見込めます。
- ファイルはアップロードされません。すべての処理がデバイス上で完結するため、動画のプライバシーは完全に守られます。
